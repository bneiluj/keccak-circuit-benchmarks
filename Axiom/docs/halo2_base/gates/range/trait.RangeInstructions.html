Returns (c, r) such that a = b * c + r.

Assumes that b != 0.

Let X = 2 ** b_num_bits. Write a = a1 * X + a0 and c = c1 * X + c0. If we write b * c0 + r = d1 * X + d0 then b * c + r = (b * c1 + d1) * X + d0
